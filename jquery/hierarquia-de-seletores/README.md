# Seletores

Na [aula passada](https://github.com/svjp/microcamp/blob/master/jquery/introducao/README.md#introdu%C3%A7%C3%A3o-ao-jquery) nós fomos introduzidos a maneira mais comum de selecionarmos um elemento DOM, que é através de um seletor do tipo *classe*, *id* ou a sua respectiva tag propriamente. Porém existem diversas outras maneiras de selecionarmos um elemento, passando alguns parâmetros em nosso seletor, podemos explicitar com detalhes qual elemento DOM queremos consultar.
